Welcome to pain-html-blog

By Sebastian Apr 1, 2016

plain-html-blog is a very clean Jekyll blog that works without CSS. It was inspired by websites which(for valid reasons)make minimal us of styling with CSS and the like.

Just to get an idea, check out stallman.org, Richard Stallman's personal website, Paulgramm.com or Mother.com (sorry for the language:). They all come with a certain 90s flavor which in 2016 is still very functional and in a way elegant(only 90s developers will know...:)

Other than that,this Jekyll blog works like any other static website so you should feel right at home.

So now go ahead, you'll want to get to the source code and read README to learn how make your own plain-html-blog.
